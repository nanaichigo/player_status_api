# player_status_api

## 環境
colorfulbox で python appでflask起動する。


## 設定
db接続は、「config.ini」で定義する。



    [DEFAULT]
    Host = xxxx
    User = xxxxx
    Password = xxxxx

    [Rugby]
    DBname = xxxxxx

|項目|内容|
|-|-|
|DEFAULT|全体設定|
|Host|接続先|
|USer|MySQLユーザ名|
|Password|MySQLパスワード|
|RUGBY|ラグビー用設定|
|DBname|データベース名|
